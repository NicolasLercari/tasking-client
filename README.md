# Run

### Docker-compose 
Follow instructions on [tasking-compose](https://github.com/NicolasLercari/tasking-compose)

### Local with NPM
Instruction:
    1. clone the repo.
```
cd tasking-client
npm i
npm start
```

**By default server running on [http://localhost:3000](http://localhost:3000)
